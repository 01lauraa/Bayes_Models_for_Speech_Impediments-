Modeling Speech Impairment with Bayesian Statistics
This project explores the application of Bayesian statistical methods to model and analyze speech impairments. By leveraging probabilistic approaches, we aim to better understand the patterns and underlying causes of various speech disorders, offering insights that can inform diagnosis and intervention.

Features
Statistical modeling of speech impairment data using Bayesian inference
Analysis and visualization of model results
Reproducible workflows for future research and clinical applications
